#WebGL – Pensumprojekt

Projektet er en WebGL-implementering udviklet som del af mit pensum.
Formålet var at arbejde systematisk med de grundlæggende koncepter i real-time
grafisk programmering og sikre korrekt forståelse af rendering-pipelinen.

Projektet er bevidst holdt afgrænset for at undgå unødvendig kompleksitet og
i stedet fokusere på fundamentale grafiske principper.

#fokusområder:
-   WebGL rendering-pipeline
-   Vertex- og fragment shaders
-   Model-, view- og projection-matricer
-   Korrekt håndtering af normaler
-   Teksturering og UV-koordinater
-   Grundlæggende scene- og kameraopsætning

#strukturen:
Projektet er opbygget med klar adskillelse mellem:
-   initialisering af WebGL-kontekst
-   shader-opsætning
-   buffer- og datahåndtering
-   transformationslogik
-   rendering-loop



Projektet gav mig en mere solid forståelse af:
- hvordan transformationer påvirker rum og orientering
- hvorfor korrekt normal-beregning er afgørende for rendering
- hvordan små fejl i pipeline hurtigt kan føre til visuelle artefakter
- vigtigheden af at holde grafiske systemer strukturerede og overskuelige


Og projektet blev udviklet efter et tidligere, mere eksperimenterende
WebGL-forsøg og repræsenterer en mere moden og fokuseret tilgang til
grafisk programmering, hvor læring og korrekthed var i centrum.

Projektet endte med at fungere som tiltænkt og gav mig et solidt greb om
de grundlæggende dele af WebGL.
